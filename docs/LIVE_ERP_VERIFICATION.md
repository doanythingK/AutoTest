## 완료 판정

아래 항목이 모두 확인되어야 완료로 본다.

- `[01/30]`부터 `[30/30]`까지 실행 로그에 남아 있다.
- 아이디/비밀번호 입력값을 자동화가 직접 수정하지 않았다.
- 로그인된 탭이 자동화 중 닫히지 않았다.
- 공급가액과 세액이 예상값과 일치한다.
- 라인 목록에 품목, 수량, 단가, 공급가액, 세액이 같은 행으로 반영되었다.
- 거래전기 완료 상태가 확인되었다.
- 회계전표입력 화면으로 이동했다.
- 원장전기 완료 상태가 확인되었다.
